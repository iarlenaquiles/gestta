# :computer: API em NodeJS de uma aplicação de CRUD de Tasks 


## Tecnologias

 - NodeJS
 - Express
 - Nodemon
 - Jest
 - Supertest
 - Sucrase
 - MongoDB
 - Mongoose
 - ESlint + Prettier
 
 
## Run
 - Git clone https://github.com/iarlenaquiles/gestta.git
 - yarn
 - run mongo service or docker
 - yarn dev
 
## Run test
  - yarn
  - run mongo service or docker
  - yarn test
